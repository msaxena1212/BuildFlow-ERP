# BuildFlow ERP - Frontend

This is the frontend application for the BuildFlow ERP system, built with Angular.

## 🚀 Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```
2.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    Navigate to `http://localhost:4200/`.

## 📦 Core Modules

The frontend is organized into the following functional modules:

*   **Dashboard:** Real-time metrics and project oversight.
*   **Projects:** Project lifecycle and detail management.
*   **Timeline:** CPM-based scheduling and visual Gantt charts.
*   **Calendar:** Daily site activity and event tracking.
*   **Team:** Personnel management and performance tracking.
*   **Access Control (RBAC):** Role-based permissions and security.
*   **Contracts:** Vendor agreement and document management.
*   **Tasks:** Kanban-style task execution tracking.
*   **Analytics & Labor:** Productivity insights and efficiency tracking.
*   **Reports:** Automated progress reporting and vault storage.
*   **Vendors:** Vendor relationship and invoicing system.
*   **Materials:** Inventory and supply chain management.
*   **Quotations:** Procurement workflow and bid management.

For a detailed end-to-end flow and feature breakdown, please refer to the [Root README](../README.md).

## 🛠️ Tech Stack

*   **Framework:** Angular (Standalone Components)
*   **Styling:** Vanilla CSS / Tailwind CSS
*   **Utilities:** RxJS, Angular CDK (Drag & Drop)
*   **Testing:** Vitest
